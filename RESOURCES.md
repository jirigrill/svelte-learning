# Svelte 5 + JS Resources

## Knowledge

- [Svelte docs (official)](https://svelte.dev/docs/svelte/overview)
  Primary source for runes, components, lifecycle, transitions. Use for: any Svelte 5 syntax question. Has an interactive tutorial too.
- [Svelte 5 tutorial](https://svelte.dev/tutorial/svelte/welcome-to-svelte)
  Step-by-step in-browser tutorial. Use for: hands-on intro to runes, snippets, effects.
- [SvelteKit docs (official)](https://svelte.dev/docs/kit)
  Routing, load functions, layouts, adapters. Use for: anything project-structure related.
- [MDN — JavaScript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)
  The canonical JS reference. Use for: JS language features, the DOM, Web APIs (IndexedDB, fetch).
- [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html)
  Use for: type annotations, narrowing, generics — when the code uses `<T>` or `as` and you want to know why.
- [Dexie.js docs](https://dexie.org/docs/)
  IndexedDB wrapper used in `atopic_helper`. Use for: queries, transactions, schema migrations.
- [The atopic_helper codebase](../atopic_helper/)
  Living example. Use for: every lesson grounds back here.

## Wisdom (Communities)

- [r/sveltejs](https://reddit.com/r/sveltejs)
  Active subreddit, official-team-friendly. Use for: questions about idiomatic Svelte 5, PWA gotchas.
- [Svelte Discord](https://svelte.dev/chat)
  Real-time help from maintainers and experienced users. Use for: stuck-on-something-specific moments.
- [Svelte Society YouTube](https://www.youtube.com/@SvelteSociety)
  Conference talks, deep dives. Use for: long-form learning when you want narrative over reference.

## Gaps
- No single "Svelte 5 for people coming from Python" resource exists yet — we'll bridge that ourselves through lessons.
